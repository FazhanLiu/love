## 🔧 自定义背景音乐

默认使用的是一个示例音乐链接。你可以替换成自己喜欢的歌曲：

1. 找一首喜欢的背景音乐（推荐网站：网易云、QQ 音乐、YouTube）
2. 获取音乐的 URL（需要支持 CORS 的源）
3. 打开 `index.html`，找到这行代码：
   ```html
   <audio id="bgMusic" loop>
       <source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg">
   </audio>
   ```
4. 将 `src` 后的 URL 替换为你的音乐链接

**推荐音乐来源：**
- 自己的音乐文件（上传到免费云存储如 GitHub、SM.MS 等）
- 网易云音乐外链
- Spotify 预览链接
- YouTube 音乐链接

## 🐛 常见问题

**Q: 音乐为什么播放不了？**  
A: 浏览器可能阻止了自动播放。点击 🎵 按钮或页面任何地方后，音乐就可以播放了。

**Q: 照片为什么上传不了？**  
A: 需要使用图片的完整 URL（网址），不能本地上传。可以先上传到免费图床（如 IMG 等）。

**Q: "不好意思"按钮为什么一直逃跑？**  
A: 说明你选的是"困难"难度，这就是功能哦！😈

**Q: 数据会不会丢失？**  
A: 清除浏览器缓存会丢失。如果要永久保存，需要有后端数据库。

**Q: 可以添加其他功能吗？**  
A: 完全可以！根据 CUSTOMIZATION.md 修改 HTML、CSS、JS 即可。
